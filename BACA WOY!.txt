!!! Tutorial Instalasi Program
-----------------------------------
1) Buka Command Prompt (CMD)

2) Arahkan CMD ke direktori dimana program ini disimpan
===> misalkan program disimpan dalam folder Documents/deployment
===> maka, lakukan perintah "CD Documents/deployment"

3) Setelah direktori berhasil diakses, ketik "pip install -r requirements.txt"
pada cmd. 

4) Tunggu proses instalasi hingga selesai.

5) Buka "environment variables" pada system komputer lalu tambahkan path folder program ini.
agar sistem cmd dapat mengakses file ffmpeg.exe, ffplay.exe, dan ffprobe.exe.

6) Instalasi selesai!
